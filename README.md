# حلقة (Halqa) — موقع تواصل اجتماعي + لوحة إدارة

موقع تواصل اجتماعي كامل مبني بـ **Node.js خالص** (بلا أي مكتبات خارجية — بلا npm install معقد)، مع قاعدة بيانات ملف JSON بسيطة. سهل تنشره فأي استضافة كتدعم Node.js.

## ✅ الميزات
- تسجيل / دخول (كلمة السر مشفرة بـ scrypt)
- بروفايل (صورة، بيو، تعديل)
- نشر منشورات (نص + صورة)، لايك، تعليقات
- متابعة / إلغاء متابعة + فيد "اللي كنتابع"
- إشعارات (لايك، تعليق، متابعة، توثيق)
- **طلب الشارة الزرقاء** من المستخدم
- **لوحة إدارة كاملة** (`/admin.html`):
  - إحصائيات عامة
  - **الموافقة أو رفض طلبات التوثيق** (الشارة الزرقاء)
  - إدارة المستخدمين (حظر / إلغاء حظر / ترقية لمشرف)
  - حذف أي منشور مخالف

## 🚀 كيفاش تخدمه محليا (على الحاسوب ديالك)

```bash
node server.js
```

بعد داك، حل المتصفح على: `http://localhost:3000`

**حساب الأدمن الافتراضي:**
- اسم المستخدم: `admin`
- كلمة السر: `admin123`

⚠️ **بدل كلمة السر ديال admin فورا** (من داخل حسابك، أو مباشرة فـ `data/db.json`) قبل ما تنشر الموقع للعموم.

## ☁️ كيفاش ترفعو على استضافة

هاد المشروع Node.js عادي، بلا build step. يمشي على:

### الخيار 1: Render.com / Railway.app (الأسهل، عندهم باقة مجانية)
1. حط المشروع فـ GitHub repo
2. فـ Render/Railway، دير "New Web Service" وربطو بالـ repo
3. Build command: (خاوي، مابغيناش)
4. Start command: `node server.js`
5. خدم متغير البيئة `JWT_SECRET` بقيمة عشوائية طويلة (سر ديالك)

### الخيار 2: VPS (Ubuntu مثلا)
```bash
# ثبت Node.js إلا ماكانش
sudo apt install nodejs npm -y

# رفع الملفات للسيرفر (scp أو git clone)
cd halqa
JWT_SECRET="حط_هنا_سر_طويل_عشوائي" PORT=3000 node server.js

# باش يبقى خدام فالخلفية، استعمل pm2:
npm install -g pm2
pm2 start server.js --name halqa
pm2 save
```
بعد داك دير reverse proxy بـ Nginx على البورت 3000 ديالك، وزيد شهادة SSL بـ Certbot.

### الخيار 3: أي استضافة كتدعم Node.js (Vercel ماشي مناسب لهاد الشكل لأنه سيرفر دائم، خدم Render/Railway/VPS/DigitalOcean/Fly.io)

## 📁 بنية المشروع
```
halqa/
├── server.js          ← السيرفر بأكمله (API + استضافة الملفات)
├── data/db.json        ← قاعدة البيانات (كتولد أوتوماتيك أول مرة)
├── public/
│   ├── index.html      ← تسجيل الدخول / حساب جديد
│   ├── feed.html        ← الصفحة الرئيسية
│   ├── profile.html     ← البروفايل
│   ├── admin.html        ← لوحة الإدارة
│   ├── css/style.css
│   ├── js/app.js, admin.js
│   └── uploads/          ← الصور المرفوعة كتخزن هنا
```

## 🔒 ملاحظات مهمة قبل ما تنشر بجد
1. **بدل `JWT_SECRET`** بقيمة سرية طويلة (متغير بيئة، ماتخليهش فالكود)
2. **بدل كلمة سر admin** الافتراضية
3. قاعدة بيانات JSON مزيانة للبداية والاختبار، ولكن إلا كبر الموقع بزاف (آلاف المستخدمين)، الأحسن تبدلها بـ PostgreSQL/MongoDB حقيقية
4. دير نسخة احتياطية (backup) دورية لـ `data/db.json` و `public/uploads/`
5. حط حد أقصى لحجم الصور إلا بغيتي (الكود فيه حد 8MB لكل صورة)
